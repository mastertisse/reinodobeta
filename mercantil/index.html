A forma mais rápida para solucionar os problemas

O SuporteOnline é um software poderoso, fácil de usar e gratuito para o acesso remoto ao computador
(via internet) tornando assim a soluçãos dos problemas se o tecnico precisar ir ate sua empresa.

Razões para usar o SuporteOnline

Acesso Seguro

O SuporteOnline usa o protocolo Remote Frame Buffer (RFB) que permite que uma área de trabalho seja visualizada e controlada remotamente pela Internet, a conexão usa uma criptografia de 128bits para total segurança dos dados na internet

Fácil Instalação

É muito fácil instalar o pragrama, basta baixar e executar o SuporteOnline.exe e pronto. Escolha um tecnico da lista e agora nossos tecnicos conseguem conectar no seu computador com o uso da internet
